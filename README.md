# GS-8_PM_Board
GS-8 Power Management Board

Master project -> [project-arrows/GS-8](https://github.com/project-arrows/GS-8)

## Overview
 * GS-8電源管理基板
![top](Hard/Picture/top.png)
![bottom](Hard/Picture/bottom.png)

## Role
 * 外部電源と内部電源の切り替えを行う。
 * 内部電源の電流・電圧を監視する。
 * 内部電源の電圧が規定値を下回った際にカットオフを行う。
 * CANバスを監視しロギングする。
